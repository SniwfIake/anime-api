Kaiju Flix API is Online! 🟢

This is your custom Cloudflare Serverless Anime API. It scrapes and bypasses CORS securely on the edge.

Available Endpoints

1. Search Anime

URL: /api/search?q=[query]

Searches Gogoanime for titles and returns IDs and cover images.

Test Search Endpoint

2. Get Anime Info & Episodes

URL: /api/info?id=[anime_id]

Extracts the hidden movie_id and pulls the raw episode list from the AJAX server.

Test Info Endpoint

3. Get Streaming Link

URL: /api/watch?id=[episode_id]

Extracts the raw video streaming embed URL securely.

Test Watch Endpoint
